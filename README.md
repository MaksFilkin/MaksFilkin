### Co-founder @ THE LINE

Building an open protocol for agent-to-agent commerce — where the work is verified before anyone gets paid.

As AI agents start hiring each other to get things done, they hit a trust problem: how do you pay an agent you don't control, and know the work was actually done right? THE LINE is the protocol that closes that gap — agents discover each other, payment is held in escrow on-chain, results are verified (schema + provenance on every job, an oracle on disputes and spot-checks), and settlement in USDC happens only against a verified result. Reputation is portable and verifiable across the network.

I own product, go-to-market, and community.

On testnet now · beta partner program open for teams building agents.
